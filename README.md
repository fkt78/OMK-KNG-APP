# カルテ略語ドリル - 看護実習の味方

看護現場で使われる医療略語を学習できるPWA（Progressive Web App）です。

## 🎯 特徴

- **300語以上の医療略語を収録**
- **クイズモード**: ランダム出題で記憶を定着
- **辞書モード**: 五十音順で検索可能（日本語・英語・略語すべて対応）
- **PWA対応**: スマホのホーム画面にアプリとして追加可能
- **オフライン対応**: ネット環境がなくても使用可能
- **完全無料**: 広告なし

## 📱 使い方

### デモサイト（GitHub Pages）

https://fkt78.github.io/OMK-KNG-APP/medical-quiz.html

### スマホのホーム画面に追加

#### iPhoneの場合（Safari）
1. 上記URLをSafariで開く
2. 画面下部の「共有ボタン」（□に↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

#### Androidの場合（Chrome）
1. 上記URLをChromeで開く
2. 「ホーム画面に追加」または「インストール」をタップ

## 📦 収録カテゴリ

### バイタルサイン
BP（血圧）、HR（心拍数）、RR（呼吸数）、BT（体温）、SpO2 など

### 疾患・病態
- **循環器**: AMI（心筋梗塞）、CHF（心不全）、AF（心房細動）など
- **呼吸器**: COPD、PNE（肺炎）、PE（肺塞栓）など
- **消化器**: GE（胃腸炎）、GERD、UC（潰瘍性大腸炎）など
- **代謝**: DM（糖尿病）、DKA など
- **腎泌尿器**: CKD、ARF、UTI など
- **脳神経**: CVA、CI（脳梗塞）、ICH（脳出血）、SAH（くも膜下出血）など
- **悪性腫瘍**: Ca、GC（胃がん）、LC（肺がん）、BC（乳がん）

### 検査
CR（胸部レントゲン）、CT、MRI、US（エコー）、ECG（心電図）、血液検査（WBC、RBC、Hb、CRP、BS など）

### 投与経路
IV（静注）、IM（筋注）、SC（皮下）、PO（内服）、SL（舌下）など

### 処置・治療
OP（手術）、ICU、CCU、ER、HD（透析）、CPR、IVH、NGT など

### 呼吸管理
O2、RA（室内気）、NC（鼻カニューラ）、各種マスク、CPAP、BiPAP、人工呼吸器など

### 薬剤
ASA（アスピリン）、APAP（アセトアミノフェン）、NSAIDs、抗生物質、ステロイド、インスリンなど

### その他
体位、創傷、整形外科、産婦人科、小児科、精神科、感染症、記号表記など

## 🛠️ 技術仕様

- **Pure HTML/CSS/JavaScript**: フレームワーク不要
- **Service Worker**: オフライン対応
- **Web App Manifest**: PWA対応
- **レスポンシブデザイン**: スマホ・タブレット・PC対応

## 📄 ファイル構成

```
.
├── medical-quiz.html   # メインアプリケーション
├── manifest.json       # PWA設定ファイル
├── sw.js              # Service Worker（オフライン対応）
├── icon.svg           # アプリアイコン
└── README.md          # このファイル
```

## 🚀 ローカルでの実行方法

```bash
# HTTPサーバーを起動（PWAのService Workerを有効にするため）
python3 -m http.server 8080

# ブラウザで開く
open http://localhost:8080/medical-quiz.html
```
